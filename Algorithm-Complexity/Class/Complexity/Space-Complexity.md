# Space Complexity
Unlike other types of [[Complexity-Index|complexity]] space complexity doesn't concern the time an algorithm takes to compute. It represents the amount of memory an algorithm uses while running.
Most algorithms have a space complexity of [[Constant-Complexity|O(1)|]]. For example [[Linear-Search|linear search]] uses the same amount of memory, regardless of the amount of inputs given. It will always only store the current index in the array that is being checked & the value to search for.
However, algorithms that "divide *&* conquer" tent to high a higher space complexity
