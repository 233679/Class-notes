# Polynomial Complexity
[[Constant-Complexity|Constant]], [[Linear-Complexity|linear]], & [[Quadratic-Complexity|quadratic]] algorithms are examples of polynomial algorithms.
The one uniting factor between these algorithms is that they all have a [[Big-O|Big O]] of O($n^k$) where $k$ is a constant value. It can be any value, even 0, as show by [[Constant-Complexity|constant complexities]].
In general these algorithms scale terribly, with O($n^0$) ([[Constant-Complexity|constant complexities]]) being the one exception.
