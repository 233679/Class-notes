Sorting algorithms are algorithms that take an unsorted array & convert it to a "sorted" array. 
There is no hard defention for what constitutes an array as sorted. It is entirely context dependent. However, a sorted array is usually regarded as a set of numbers in assending order or a collection of words in alphabetical order.
